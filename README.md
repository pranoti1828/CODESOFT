# CODSOFT Java Development Internship

## Task 1: Number Game

This is a simple Java console-based number guessing game.

### Features:
- Generates a random number between 1 and 100
- Takes user input
- Provides feedback (Too High / Too Low / Correct)
- Repeats until the correct number is guessed

### Language Used:
- Java

### How to Run:
```bash
javac NumberGame.java
java NumberGame
